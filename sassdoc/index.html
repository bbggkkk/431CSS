    maxLength

    $maxLength: 1000;

    Description

    비 컴포넌트 CSS가 생성될 범위를 지정합니다. 해당 CSS는 0부터 시작하는 특정 조건에 해당하는 값을 가집니다.

    Example

    범위 지정을 활용한 tblr 함수 사용

    @for $i from 0 to $maxLength {
    @if $i%2==0 or $i%5==0 {
        @include tblr("margin","m",$i);
        @include tblr("padding","p",$i);
    }
}

    mixins

    tblr

    @mixin tblr($nm, $css, $val) { ... }

    Description

    방향(top,bottom,left,right)을 지정할 수 있는 CSS에 각 방향을 붙여줍니다. @inlcude하면 클래스명이 반환됩니다.

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $nm

    클래스 이름으로 사용할 이름을 지정합니다.

    		String— none
    $css

    클래스에서 사용할 CSS를 입력합니다.

    		String— none
    $val

    CSS에 들어갈 값을 입력합니다.

    		Number— none
